# CRAFT SMS 🚀
### Unified Educational Platform for Emerging Markets

**CRAFT SMS** is a high-security, multi-tenant School Management System designed specifically to bridge the digital gap in regions with unstable connectivity. By combining a robust **Python (FastAPI)** vault with **Offline-First (PWA)** technology, CRAFT ensures that education management never stops, even when the internet does.

## 🌟 Key Features
- **Multi-Tenant Architecture:** Secure data isolation for multiple schools on a single infrastructure.
- **Offline-First Sync:** Proprietary sync manager allowing teachers to work offline and auto-sync when 2G/3G returns.
- **Identity & Security:** Automated E-ID generation with functional barcodes and server-side verification.
- **Financial Vault:** Manual bank-slip verification loop for accurate tuition tracking.
- **Gamified Learning:** Real-time student leaderboards to drive engagement and performance.

## 🛠️ Tech Stack
- **Backend:** Python (FastAPI)
- **Database:** PostgreSQL (Supabase)
- **Frontend:** Next.js + Tailwind CSS
- **Mobile:** PWA (Progressive Web App) with Capacitor Wrapper
- **Documentation:** ReportLab (Python-based PDF generation)

## 🌍 The Vision
Starting in Liberia, CRAFT SMS aims to digitize the entire educational infrastructure of West Africa, moving from manual paper records to a secure, unified digital ecosystem.
